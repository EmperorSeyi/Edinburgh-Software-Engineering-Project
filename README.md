## ğŸ“˜ Project Overview

This repository contains the implementation and documentation for the **Scotch Broth**
software development project, delivered for Creative Transformation Support (CTS).

The long-term product vision is a **collaborative storytelling platform** for building
shared worlds, characters, and timelines.

Given the fixed time and budget constraints of a single-semester academic project,
the team intentionally focused on delivering a **small, high-quality MVP prototype**
rather than a complete production system.

---

## ğŸ¯ MVP Scope

The implemented MVP demonstrates a focused **narrative engine for character
development over timelines**.

The prototype supports:

- Creating characters  
- Creating events  
- Applying attribute changes over time  
- Computing character state at any point in the timeline  
- Exporting a static HTML summary for review  

The MVP is implemented as a **Python-based CLI application**, with persistence via
MongoDB and full environment reproducibility using Docker.
